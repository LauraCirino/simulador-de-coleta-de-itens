# Simulador de coleta de itens

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
![Status-Concluído](https://img.shields.io/badge/Status-Concluído-brightgreen?style=for-the-badge&logoColor=white)

Uma pagina que simula a coletas de itens atraves do clique de um botão

---
## 📚 Aprendizados
- loop while


## ⚙️ Funcionalidades
- cada clique coleta um item
- ao coletar 10 itens o botão e desabilitado e aparece uma mensagem na tela


## 🛠️ Stack
- JavaScript
- HTML5


## 🚀 Deploy
1. Clone ou baixe este repositório.

```bash
git clone https://github.com/LauraCirino/simulador-de-coleta-de-itens
```

2. Abra o arquivo `index.html` em seu navegador.
